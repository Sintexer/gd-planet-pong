# Planet Pong

[<img src="/Screenshots/planetPongThumbnail.png">]([https://link-to-your-URL/](https://ilboogl.itch.io/planet-pong))
  
**Game is available at [itch.io](https://ilboogl.itch.io/planet-pong)**

**Planet Pong** is a modern reimagining of the grandfather of video games. I took the classic loop and injected it with Juice to create an experience that feels as heavy as a collapsing star. This is my way of saying "Hello World!" to the itci.io community.

> This isn't just about hitting a ball; it's about surviving the impact.
